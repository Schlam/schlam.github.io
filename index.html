Welcome to my website

Here are some things I've worked on

PyShift

A python wrapper for the Pushshift.io API. Can be used to generate rich datasets, collect topically labeled text, create time-series signals or monitor keyword occurances in social media interactions in real time.

Bitcoin Forecasting

Training a neural network to predict the future behavior of Bitcoin. Trained using historical data only, reducing potential for adversarial attack relative to its keyword/sentiment-aware precursor.

Identifying bird species by fine tuning an InceptionV3 model

Fine-tuning a pretrained image classification model (InceptionV3) to recognize bird species from the Caltech Birds dataset, and using augmentation techniques to overcome data scarcity.

Data Engineering, Natural Language Processing, and Analysis of Airbnb listings and reviews in Rio de Janeiro

A data transformation pipeline that runs completely on the cloud, using Apache Spark dataframes and AWS database and file-hosting services. Language detection is performed on a massively multilingual corpus of user reviews using SOTA models from Spark NLP, along with exploration and analysis of data from the relevant Airbnb listings

Semantic Classification Pipeline with Sci-Kit Learn

Performing Latent Semantic Analysis (LSA) and training a classifier using posts (documents) from Reddit. LSA is a technique for embedding text using frequencies of word combinations (terms), and operates under the assumption that statistical regularities arise when analyzing a reasonably large corpus. Logistic regression is used to learn a mapping between our latent space and the appropriate label, which in our case is the subreddit from which our document was retrieved.

r/Suggest

A simple web interface for generating forum suggestions from user input using this model. Returns most closely related forum on Reddit from a list of 10.

Custom training loop for Few-Shot Object Detection

Scripts for custom training loop. Rebuilding an SSD ResNet50 model from a tensorflow model checkpoint, taking gradients with respect to the bounding box loss only

Image Segmentation with Caltech Birds

Training scripts for an image segmentation experiment I performed, which combined ideas from two well known papers in computer vision. A key element of this experiment was building a UNet from a VGG19 base architecture. Another focus was the data-- which I had prior experience working with at the time. High variance in the distribution of feature sizes present in our images motivates the desire for a residual network in place of a deep sequential model.
